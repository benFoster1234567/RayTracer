 Whitted Ray Tracer

==================

A CPU driven whitted-style ray tracer written from scratch using **C++20**


## Key Highlights

*  **Recursive Reflections:** Supports multiple bounces.

*  **Zero Dependencies:** Relies only on C++'s standard template library. Custom linear algebra library

*  **Test-Driven Development:** Developed using a testing suite.


## What's Next

- [ ] **Anti-Aliasing:** Implementing jittered super-sampling for smoother edges.

- [ ] **OBJ Support:** Parsing 3D models via custom `.obj` reader.

- [ ] **YAML Scene Parsing:** Moving from hardcoded scenes to dynamic configuration files.
